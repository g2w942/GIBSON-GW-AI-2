## 🚀 DEPLOYMENT TO GITHUB PAGES — Step by Step

### Prerequisites
- GitHub account ([sign up free](https://github.com/signup))
- Git installed on your computer ([download](https://git-scm.com/downloads))
- OR use GitHub's web interface (no Git needed)

---

### METHOD 1: GitHub Web Interface (Easiest — No Git Required)

#### Step 1: Create GitHub Account
1. Go to https://github.com
2. Click "Sign up"
3. Enter email, create password, choose username
4. Verify your email

#### Step 2: Create New Repository
1. Click the "+" icon (top-right) → "New repository"
2. Repository name: `gibson-gw` (or any name you prefer)
3. Description: `GIBSON GW — EDITH Self-Evolving AI`
4. Select **Public** (required for free GitHub Pages)
5. ✅ Check "Add a README file"
6. Click "Create repository"

#### Step 3: Upload the 4 Files
1. Inside your new repo, click "Add file" → "Upload files"
2. Drag and drop these 4 files:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `README.md`
3. Scroll down
4. Commit message: "Initial commit — GIBSON GW EDITH v2.0"
5. Click "Commit changes"

#### Step 4: Enable GitHub Pages
1. Go to your repo → **Settings** (top menu)
2. Click **Pages** (left sidebar)
3. Under "Source":
   - Branch: select **main** (or **master**)
   - Folder: select **/ (root)**
4. Click **Save**
5. Wait 1-2 minutes
